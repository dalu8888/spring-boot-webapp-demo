How to run ?

1. fire create table sql script in dbscripts folder.
2. change the datasource setting in /src/main/resource/application.yaml
3. load source into idea15.
4. application > boot run, also you can build the project with gradle command.
5. access http://localhost:8080/book, it will rout to login page, you can login with user/password, you will see the page same as screenshot (online.png)
